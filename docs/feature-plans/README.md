# Feature Plans

This directory contains planned features for Voice Notepad. These are public design documents describing features under consideration or in development.

## Status Legend

- **Planned** - Designed but not started
- **In Progress** - Currently being implemented
- **Completed** - Merged and released
- **Deferred** - Postponed for later consideration

## Current Plans

| Feature | Status | Description |
|---------|--------|-------------|
| [Semantic Search](semantic-search.md) | Planned | Find transcripts by meaning using Gemini embeddings |
